# MARL Dynamic Pricing Prototype

## Описание
Прототип системы для моделирования динамического ценообразования в олигополии с использованием многопользовательского обучения с подкреплением (MARL).  
Фирмы – X5, Магнит, Лента. Спрос описывается logit-моделью.

## Требования
- Python 3.8+
- Установить зависимости: `pip install -r requirements.txt`

## Структура репозитория
- `main_code.ipynb` – основной ноутбук с кодом
- `data.ipynb` – данные компаний
- `requirements.txt` – список зависимостей
- `README.md` – этот файл

## Запуск
1. Открыть `main_code.ipynb` в Jupyter Notebook.
2. Выполнить ячейки последовательно.
3. После завершения обучения будут выведены метрики и построены графики.

## Основные компоненты:

- `OligopolyMarketEnv` — среда олигополии с logit-спросом и инерцией спроса.
- `DQNAgent` — независимый DQN-агент с буфером воспроизведения и target-сетью.
- `ReplayBuffer` — буфер для хранения опыта.
- `DQNNetwork` — нейросеть для аппроксимации Q-функции.
